# 🌌 Planetarium & Observatory Management System

## UI/UX Design Internship Project – NCGSA

This repository contains the research, documentation, and UI/UX design work completed as part of my internship at the **National Center of GIS & Space Applications (NCGSA)**.

The project focuses on designing a modern, user-centered **Planetarium & Observatory Management System** that provides an intuitive experience for visitors, students, researchers, faculty, and administrators. The design process follows a structured UX workflow, beginning with research and user analysis before progressing to wireframes, visual design, and interactive prototypes.

---

## 📌 Project Objectives

- Research user needs and identify target audiences.
- Analyze existing planetarium and observatory websites.
- Design an intuitive information architecture and user flows.
- Create modern, accessible, and visually engaging UI designs.
- Develop interactive prototypes for key user journeys.
- Maintain a structured design process with supporting documentation.
